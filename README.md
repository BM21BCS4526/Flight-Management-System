Overview

The Flight Management System is a backend application designed to manage flight operations such as flight scheduling, passenger bookings, and airline administration.
It provides REST APIs to handle flight data, bookings, and user interactions.
The system focuses on modularity, scalability, and efficient data handling.

Features

Flight creation and management
Flight search based on source, destination, and date
Passenger booking system
Seat availability tracking
Booking cancellation
Admin controls for managing flights
RESTful API architecture

Tech Stack

Backend
Java / Spring Boot (or Node.js / Express if applicable)
Database
MySQL / PostgreSQL

Tools

Maven 
Postman (API testing)
